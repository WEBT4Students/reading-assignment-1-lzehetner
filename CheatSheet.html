World Wide Web General

  1. What are the most essential technologies that made the World Wide Web such a success?
    HTML, HTTP(S), URL

  2. How are resources (html pages, images, etc) addressed in the web?
    with URI(uniform recource identifier). It locates the adress or information.

  3. What are the general components of a url?
    • A scheme: indicates the internet resource's access protocol
    • A host: the resource's host is identified by this information
    • A path: indicates the particular host resource that the web client wishes to access
    • A query string: gives a string of data for the resource and then follows the path component

Web Server

  1. What is a web server?
    A web server is computer software and underlying hardware that accepts requests via HTTP or its secure variant HTTPS

  2. What does a web server?
    In essence, a web server is simply a powerful computer that stores and transmits data via the internet

  3. How can content be stored on a web server?
    They are frequently generated dynamically by server-side code rather than being explicitly stored, and a reverse proxy may even cache them

  4. How is content in webpages described?
    Web content refers to the textual, aural, or visual content published on a website

Web Client

  1. What is a web client?
    A web client is a client-side application used for connecting to a web server over HTTP

  2. What does a web client?
    It displays web pages received from the server and allows users to interact with the web server
